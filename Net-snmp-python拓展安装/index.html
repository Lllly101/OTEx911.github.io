记笔记-SIP和SNMP-python拓展安装

Posted on 2017-05-18 | In Others

OS X EI Capitan上有个系统完整性保护技术(SIP-System Integrity Protection)，旨在帮助防止潜在恶意软件修改 Mac 上受保护的文件和文件夹。系统完整性保护可以限制 root 用户帐户，以及 root 用户能够在 Mac 操作系统的受保护部分执行的操作。所以有些时候，你需要关闭SIP。

关闭SIP

开机，command+R进入恢复模式，打开terminal，输入csrutil命令。根据提示进行关闭或者启用SIP。

net-snmp安装

Mac本身自带了net-snmp，但并没有提供python的扩展。尝试用pip和easy_install去安装，仍然无果。只能源码安装。

net-snmp-5.6.2

下载完成后，进行编译、测试、安装即可。
