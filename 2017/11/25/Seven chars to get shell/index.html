The reason that we created these file in a reverse order is I can only use seven chars. Command ls >1 output the filenames by default order instead of created time. Therefore I need this one ls -t>1. Output is the following.

Please pay attention to the \whitespace and \\. In shell, whitespace should be escaped and so is the backslash. I used the backslash to split command over multiple lines.
